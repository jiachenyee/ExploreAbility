# ExploreAbility
## a slightly overengineered way of teaching iOS Accessibility 

## frameworks & technologies
### client app
- **SwiftUI** 
- **UIKit** 
- **CoreLocation**
- **AVFoundation/AVKit**
- **MultipeerConnectivity**
- **UserNotifications** 
- **ActivityKit/WidgetKit** 
- **CoreHaptics**
 
### console app 
- **SwiftUI** 
- **SceneKit**
- **MultipeerConnectivity**
- **CoreBluetooth**
- **CoreLocation**
- **MapKit**

### room scanner
- **UIKit**
- **RoomPlan**

### iBeacon (Mac/iPad)
- **SwiftUI** 
- **CoreBluetooth**
- **CoreLocation**

> Designed for the Apple Developer Academy @ Infinite Learning.
